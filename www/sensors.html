Sensors

  • Ultrasonic anemometer, capable of measuring wind velocities in 3 dimensions
  • Cup anemometer measures wind magnitudes
  • Wind Vane measures wind direction on the horizontal plane
  • Embedded Temperature Sensor
  • Rain Gauge
  • Humidity Sensor

Sensor Interfacing

Ultrasonic anemometer can be interfaced to the Raspberry Pi via an RS232 Driver/Receiver

Cup anemometer, wind vane, raing gauge, humidity and temperature are connected via an expansion board to a Waspmote Pro Board. The Waspmote compiles the values and sends that data as serial data via USB to a Raspberry Pi
