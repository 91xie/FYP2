Central Hub

A Raspberry Pi is used as a central hub. LAMP installation was completed on the central hub. LAMP stands for Linux, Apache, MySQL, PHP.

A python script was written to read the incoming data from the remote weather station, process it and put it the database.

MySQL is the RDBMS used to store the weather data.

Apache is the web server software used to host the website.

PHP is used to the pull information from the MySQL database and display on the website.
